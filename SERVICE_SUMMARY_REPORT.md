# [보고서] 내일(Naeil) 무장애 축제 지도 서비스 기능 총정리

**작성일**: 2026년 4월 17일  
**대상**: 프로젝트 운영위원회 및 주요 이해관계자  
**시스템명**: 내일(Naeil) - 무장애 축제 정보 플랫폼

---

## 1. 개요
'내일'은 교통약자(장애인, 노인, 유모차 이용자 등)가 전국 축제를 물리적 제약 없이 즐길 수 있도록 돕는 **실측 데이터 기반 무장애 지도 서비스**입니다. 본 보고서는 소비자(사용자)와 관리자의 관점에서 구현된 핵심 기능을 정리한 내용입니다.

---

## 2. 소비자(사용자) 관점 주요 기능
현장에서의 편의성과 정보의 신뢰성을 최우선으로 설계되었습니다.

### 📍 실측 기반 인터랙티브 지도 (Interactive Map)
*   **고해상도 축제 안내도**: 축제 공식 안내도 위에 배리어프리 정보를 레이어로 결합하여 제공합니다.
*   **지능형 핫스팟**: 화장실, 입구, 도움 센터 등 주요 지점을 클릭하여 상세 정보를 확인할 수 있습니다.
*   **실제 현장 사진 슬라이더**: '텍스트'가 아닌 '사진'으로 미리 현장을 확인할 수 있어 가보기 전 불안감을 해소합니다.
*   **직관적 픽토그램**: 휠체어 단독 가능 여부, 경사로 유무 등을 아이콘으로 한눈에 파악할 수 있습니다.

### 🗓️ 일정 및 정보 탐색
*   **축제 캘린더**: 월별 축제 일정을 바(Bar) 형태의 시각적 인터페이스로 한눈에 파악합니다.
*   **상세 스토리**: 축제의 취지와 배리어프리 준비 과정을 사용자에게 공유하여 문화적 공감대를 형성합니다.

### 📢 참여형 신고 센터 (Report System)
*   **실시간 불편 제보**: 사용자가 현장에서 겪은 불편함을 즉시 사진과 함께 제보할 수 있습니다.
*   **지도 기반 위치 핀(Pin)**: 지도 위를 직접 클릭하여 불편함이 발생한 정확한 지점을 관리자에게 전달합니다.
*   **자동 데이터 압축**: 현장에서 저속 네트워크 환경에서도 원활하게 사진이 업로드되도록 자동 최적화를 지원합니다.

### 📱 최적화된 사용자 경험 (UX/UI)
*   **반응형 웹 디자인**: 모바일(현장 사용), 태블릿, 데스크탑 등 모든 기기에서 최적화된 화면을 제공합니다.
*   **초고속 로딩**: 5MB 이상의 대용량 이미지도 시스템 내부에서 품질 저하 없이 압축하여 즉각적인 로딩 속도를 보장합니다.

---

## 3. 관리자 관점 주요 기능
직관적인 콘텐츠 관리와 안정적인 데이터 운영에 초점을 맞추었습니다.

### 🔐 보안형 관리 패널
*   **전용 로그인 시스템**: 승인된 관리자만 접근 가능한 독립된 관리 환경을 제공합니다.
*   **간편 진입**: 일반 사용자에게 노출되지 않는 푸터의 비밀 링크를 통해 빠르게 관리 화면으로 로그인할 수 있습니다.

### 🏗️ 콘텐츠 라이프사이클 관리 (CMS)
*   **축제 등록/수정/삭제**: 축제 기본 정보, 기간, 장소 및 대표 이미지를 간편하게 관리합니다.
*   **이미지 최적화 엔진**: 관리자가 대용량 파일을 업로드해도 서버 부하를 줄이기 위해 웹 최적화가 프로세스가 자동 실행됩니다.

### 🎯 정밀 핫스팟 에디터 (Hotspot Editor)
*   **위지윅(WYSIWYG) 방식**: 지도 이미지를 직접 클릭하여 핫스팟 위치를 지정하는 직관적 인터페이스를 제공합니다.
*   **정밀 좌표 제어**: X, Y 좌표 및 크기(W, H)를 수치로 직접 조정하여 지형지물에 완벽히 일치시킵니다.
*   **스마트 텍스트 변환**: 간단한 줄바꿈만으로도 사용자 화면에서는 목록형(•) 디자인으로 자동 변환되는 편의 기능을 탑재하였습니다.
*   **실시간 미리보기**: 편집 중인 내용이 사용자에게 어떻게 보일지(팝업 모달 포함) 별도의 저장 없이 즉석에서 확인 가능합니다.

### ☁️ 클라우드 기반 데이터 운영
*   **실시간 동기화**: Firebase 연동을 통해 관리자의 수정 사항이 전 세계 사용자에게 실시간으로 반영됩니다.
*   **데이터 안정성 보장**: 클라우드 분산 저장을 통해 브라우저 캐시 삭제 시에도 데이터가 유실되지 않으며 강력한 보안성을 가집니다.

## 5. PDF 추출 방법 안내
이 문서를 PDF로 저장하려면 다음 단계 중 하나를 선택하세요.

1.  **브라우저 이용 (가장 추천)**:
    *   현재 화면(또는 파일의 웹 미리보기)에서 `Ctrl + P` (Windows) / `Cmd + P` (Mac)를 누릅니다.
    *   대상(Destination)을 **'PDF로 저장(Save as PDF)'**으로 선택하고 저장합니다.
2.  **마크다운 툴 이용**:
    *   VS Code 사용 시: `Markdown PDF` 확장 프로그램을 설치 후 마우스 우클릭 -> `Export (pdf)`를 선택합니다.
